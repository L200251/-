# 李佳瑶 · 深深 个人简历网站

## 运行方式

```bash
npm install
npm run dev
```

浏览器打开：

```bash
http://localhost:3000
```

## 修改内容

主要修改 `app/page.tsx` 顶部的 `assets` 和 `siteData` 即可。
